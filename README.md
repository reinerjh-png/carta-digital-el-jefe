## 📖 Descripción

**El Jefe Lounge** es una carta digital web premium diseñada para un restobar amazónico con ambiente nocturno. Esta no es una landing corporativa tradicional, es una **experiencia gastronómica digital** que combina:

- 🌿 Estética de selva amazónica + nightlife de lujo
- 💚 Tema dark mode con verde neón (#22C55E) y acentos dorados
- ✨ Animaciones inmersivas y micro-interacciones
- 📱 Diseño mobile-first optimizado para QR en mesas

### Experiencia Visual
- [ ] Preloader animado con anillos giratorios
- [ ] Sistema de partículas flotantes

## 🚀 Instalación y Uso

### Opción 1: Local
```bash
# Clona o descarga el proyecto
cd ElJefe

# Abre directamente en tu navegador
open index.html
```

### Opción 2: Live Server (Recomendado)
```bash
# Con VS Code, instala la extensión "Live Server"
# Click derecho en index.html → "Open with Live Server"
```

### Opción 3: Deploy en Vercel
```bash
# 1. Sube el proyecto a GitHub
git init
git add .
git commit -m "Initial commit"
git remote add origin <tu-repo>
git push -u origin main

# 2. Conecta tu repo en Vercel
# Visita https://vercel.com y sigue las instrucciones
```

## 🔧 Personalización

### Cambiar Color Principal
```css
:root {
    --neon-green: #TU_COLOR;
}
```

### Modificar Menú
Edita los `<article class="menu-card">` en `index.html`:
```html
<article class="menu-card" data-category="platos">
    <div class="card-glow"></div>
    <div class="card-image">
        <span class="card-emoji">🍝</span>
    </div>
    <div class="card-content">
        <h3 class="card-title">Nombre del Plato</h3>
        <p class="card-desc">Descripción</p>
        <div class="card-footer">
            <span class="card-price">S/ 00</span>
            <button class="card-btn">+</button>
        </div>
    </div>
</article>
```

### Cambiar WhatsApp
Edita el href en `index.html`:
```html
<a href="https://wa.me/51999999999" class="whatsapp-float">
```
